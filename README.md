# Mocko Language Test Blog

This repository contains a simple GitHub Pages website created to introduce Mocko's AI-powered language mock test resources.

## About Mocko

Mocko is an AI-powered language learning platform that helps candidates prepare for international language proficiency exams through realistic mock tests, detailed performance analysis, and personalized learning insights.

The website includes introductory pages for:

* CELPIP Mock Test
* TEF Mock Test
* TCF Mock Test
* Versant Mock Test
